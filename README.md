Description:
A compact personal project created to enhance programming proficiency and gain hands-on experience with dependency injection and common game architecture patterns in Unity.

How to run the project:
1) Open the project in Unity
2) Add the Menu scene
3) Press Play

Technologies and patterns used:

--Unity Engine 2022.3.9f1
--C#
--Zenject (DI, SignalBus)
--JSON-based
--Assembly Definitions
--UniRx
--MVVM architecture
--UniTask
--Yandex Ads
--Design Patterns: Factory, Object Pool, Strategy
